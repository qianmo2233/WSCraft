# 欢迎使用 WSCraft

**WSCraft 是一个为Bukkit打造的WebSocketAPI**

如果您想要在您的网站上实时显示服务器的一些信息，或是让玩家在网页或其他客户端上与服务器互动，那么你就需要它！


## 如何使用？
**For服主**
直接丢进Plugins文件夹即可
按需修改配置文件：
`config.yml`
```yaml
#WebSocket监听端口
Port: 8888
#是否开启DEBUG模式
DEBUG: false
```
